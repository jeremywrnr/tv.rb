`tv.rb` (maybe) emulates a retro tv's fuzzy flashing.

otherwise just some trippy terminal art that's fun.

by jeremy warner, december 2015


## setup

    $ [sudo] gem install tv.rb


## usage

    $ tv.rb

for more waves:

    $ tv.rb --wavy

Then, press ctrl-c to quit.
